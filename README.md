KUnnect

[서비스 개요]
KUnnect는 고려대학교 정보대학 학생들을 위한 포트폴리오 관리 및 경험 데이터 기반 챗봇 서비스입니다. 전공 과목에서 우수한 성적을 받는 것만으로는 실무 세계로 연결되지 않는 현실 속에서, 학교와 실무 사이의 간극을 메우는 실질적인 성장 가이드를 제공합니다.

[핵심 문제 인식]
- 전공 성적 ≠ 실무 준비: 4.5 만점을 받아도 포트폴리오 경험, 협업 과정, 기술 스택 등 실제 취업에 필요한 것들은 별도로 준비해야 함
- 졸업생/현업 선배들의 실제 경험 부족: 학교에서 배우는 내용과 현실 세계의 실무 사이 괴리를 어떻게 메울지 막막함
- 범용 GPT의 한계: 고려대학교 정보대학이라는 특수한 맥락과 선배들의 구체적 경험을 반영하지 못함

[핵심 가치 제안]
고려대학교 정보대학만의 특성, 커리큘럼, 교수진, 동문 네트워크를 활용하여, 지금 무엇을 해야 하는지 구체적이고 실행 가능한 가이드를 제공합니다.

[주요 기능]
1️⃣ AI 챗봇 (경험 기반 질의응답)
진로, 수업, 프로젝트, 취업/인턴 등 실제 고민에 대한 선배들의 경험을 기반으로 답변
챗봇 응답 시 출처(글)와 작성자 프로필 제공 → 1대1 연결 가능
RAG(Retrieval-Augmented Generation) 기반으로 신뢰성 있는 정보 제공

2️⃣ 포트폴리오 관리
자신의 기술 스택, 업무 경험, 프로젝트, 활동 및 수상 경력 등을 체계적으로 관리
공개/비공개 설정 가능 (공개 시 다른 사용자들에게 참고 자료로 활용)

3️⃣ 경험 공유 피드
학교 생활 팁, 수업 후기, 프로젝트 경험, 취업/인턴 후기 등을 카테고리별로 작성/공유
카테고리: 진로/공부 방향, 수업/과목, 프로젝트/활동, 취업/인턴, 기타
커뮤니티 형성을 통한 지식 축적 및 선순환 구조

[확장 가능성]
📚 학교 내부 확장
단과대 확장
정량/정성 데이터: 커리어/인턴 정보, 공유 활성화

🏢 산업/기업과의 연계
산학 협력 프로그램 자동화
학기 단위 선택 프로젝트 관리 기능
대학 기반 산업 채용 파이프라인 구축 가능

🤖 AI 중심 서비스 확장
전공/커리어 데이터 생성 → 더 정교한 추천
특정 진로 희망 시 이수 전공 조합, 추천 고도화
이론 전공 조합 → 추천 진로 '알고리즘' 고도화

💾 데이터 기반 서비스 확장
대학 커리어 데이터 리포트 발행
어떤 과목/경험/위치/연구 이어졌는지 대학 성과 분석 자료 제공
학과가 학생의 실무 역량 추적 가능
